# Valentine Web Experience

A cinematic interactive Valentine's Day web experience built with **React + TypeScript**.
Designed as a multi‑page emotional journey combining animation, music, and visual storytelling.

---

##  Overview

This project is a single‑page application that presents a four‑scene Valentine's experience:

1. **Introduction Page** – A gentle welcome with cosmic starfield atmosphere.
2. **Message Page** – Romantic text designed to feel personal and heartfelt.
3. **Music Page** – Embedded YouTube mini‑player with thumbnail overlay controls.
4. **Final Page** – A closing Valentine's message celebrating love and connection.

The site uses smooth page transitions, animated hearts, and a starfield background to create an immersive atmosphere.

---

##  Features

### Smooth Cinematic Transitions

Pages fade and slide between scenes to create a soft storytelling flow.

### Interactive Music Player

A YouTube‑powered music player with:

* Thumbnail display
* Dark overlay controls
* Play/pause interaction

### Animated Starfield Background

A dynamic space backdrop adds depth and ambience throughout the experience.

### Heartbeat Animations

Custom SVG heart elements gently pulse across the interface.

### Responsive Layout

Designed to remain visually consistent across screen sizes.

##  Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/valentine-web-experience.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
